📝 НУЖНО СДЕЛАТЬ (3 шага):

1️⃣  УКАЗАТЬ ID ТАБЛИЦЫ
    
    Откройте вашу Google таблицу в браузере
    URL: https://docs.google.com/spreadsheets/d/[ЭТО_ID]/edit
    
    Скопируйте ID и вставьте в файл .env:
    GOOGLE_SPREADSHEET_ID=ваш_id_таблицы

───────────────────────────────────────────────────────────

2️⃣  ПОЛУЧИТЬ КЛЮЧ ПОГОДЫ
    
    1. Зайти на https://openweathermap.org/
    2. Зарегистрироваться (бесплатно)
    3. Скопировать API key
    4. Вставить в .env:
       OPENWEATHER_API_KEY=ваш_ключ

───────────────────────────────────────────────────────────

3️⃣  НАСТРОИТЬ GOOGLE API
    
    1. Зайти на https://console.cloud.google.com/
    2. Создать проект
    3. Включить API:
       - Google Sheets API
       - Google Drive API  
       - Generative Language API
    4. Создать OAuth credentials (Desktop app)
    5. Скачать JSON → переименовать в credentials.json
    6. Положить в папку credentials/
    
    Подробно: см. credentials/README.md
